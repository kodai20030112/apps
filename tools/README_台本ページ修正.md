# 台本ページ修正ツール（おぼえりふ用）

AI（Claude等）に台本PDFをExcel化してもらうと、**「ページ」列が見開きの境目で1ページずれる**ことがあります。
このツールは元のPDFを直接読み、各セリフが**実際に印刷されているページ**を割り出して、ページ列を正しく振り直します。

## 毎回の手順（あなただけでできます）

1. いつもどおり、Claude に台本PDF＋テンプレートを渡してExcelを作ってもらう
   （おぼえりふの「取り込み」画面のコピー用プロンプトを使う）
2. **`台本ページ修正.command` をダブルクリック**
3. 出てきたウィンドウに、聞かれた順に
   - ① 台本の **PDF** をドラッグ → Enter
   - ② AIが作った **Excel(.xlsx)** をドラッグ → Enter
4. 同じフォルダに **`〜_ページ修正版.xlsx`** ができる
5. それを **おぼえりふで取り込む** → ページごとに正しく表示されます

## 仕組み（かんたんに）

- PDFの各ページ下に印刷された「-12-」のようなページ番号を読み取る
- 見開き（1枚に2ページ）の場合は左右に分けて、縦書き（右→左）の読み順で本文を復元
- Excelの各セリフを、その本文と上から順に突き合わせて、印刷ページ番号を割り当てる
- ページをまたぐセリフは「始まったページ」に統一する

## 結果の見方

実行後に出る数字：

- **ページを直した行**：AIがずらしていて直した数
- **ページが逆戻りした箇所：0** … 0ならページの整合が取れている合図。
  0より大きいときは、PDFのページ番号がうまく読めていない可能性があります。

## 前提・うまくいかないとき

- 縦書き（右→左）の日本語台本PDFを想定しています。
- **各ページの下にページ番号が印刷されている**ことが必要です（手書きスキャン等で番号が無いと振り直せません）。
- Excelは「番号／ページ／登場人物／セリフ」の並び（おぼえりふのテンプレート）であること。
- ライブラリ（pymupdf, openpyxl）は `.command` が初回に自動で入れます。

## 中身

- `fix_serifu_pages.py` … 本体。`python3 fix_serifu_pages.py <PDF> <Excel> [出力]` でも直接使えます。
- `../台本ページ修正.command` … ダブルクリック用（ドラッグ＆ドロップで実行）。
